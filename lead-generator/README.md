# Lead Generator — Multi-Platform B2B Data Extraction Tool

> Extracts verified business leads — phone numbers, emails, ratings, addresses — from JustDial, IndiaMart, TradeIndia, and Google Maps into structured Excel or PDF output.

## Overview

A desktop automation tool that scrapes structured business contact data from major Indian B2B platforms. Users enter keywords and location, the tool extracts verified leads with contact details, and exports them to Excel or PDF — ready for outreach.

---

## Problem Solved

Manual lead research from B2B directories is slow, inconsistent, and impossible to scale. This tool automates the extraction process across four platforms simultaneously, returning clean, structured data in minutes.

---

## Features

- **Multi-platform extraction** — JustDial, IndiaMart, TradeIndia, Google Maps in one tool
- **Dynamic input** — keyword and location configurable per session via GUI
- **Data extracted** — Company name, ratings, phone number, email, full address, and more
- **Automated OTP authentication** — IndiaMart login handled automatically via Selenium
- **Excel export** — customizable formatting and filtering options
- **PDF export** — formatted lead reports for sharing
- **Desktop GUI** — Tkinter interface with custom icons and intuitive UX
- **Session stability** — bug-fixed and tested for extended scraping runs

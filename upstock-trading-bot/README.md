## Overview

An automated trading bot that connects to the Upstox API, processes live market data in real time, applies configured trading logic, and executes buy/sell orders autonomously — with no human intervention required during operation.

---

## Problem Solved

Manual trading requires constant market monitoring and fast execution — both difficult to maintain consistently. This bot applies a defined strategy with zero reaction delay and operates continuously without fatigue.

---

## Features

- **Live market data processing** — real-time price feed via Upstox WebSocket API
- **Autonomous order execution** — buy/sell orders placed without manual trigger
- **Configurable strategy** — trading parameters adjustable without code changes
- **100% uptime reliability** — operated continuously in live production environment
- **Error handling** — connection drops and API failures handled with automatic recovery
